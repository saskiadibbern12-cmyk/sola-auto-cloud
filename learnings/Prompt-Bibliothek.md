# Prompt-Bibliothek für @sola.makes

> Wiederverwendbare Prompts, abgeleitet aus deinen gespeicherten Reels und angepasst auf dein **KI-Creative-Studio für Beauty & Wellness**.
> Einfach kopieren, `[Platzhalter]` füllen, in Claude / das jeweilige Tool einsetzen.

---

## 1. Cineastisches Beauty-Editorial (Bild) , Stil „Higgsfield Soul"

```
Du bist Art Director für eine High-End-Beauty-Kampagne.
Erzeuge ein fotorealistisches Editorial-Bild:

Produkt/Subjekt: [z.B. Serum-Flasche, bernsteinfarben]
Stimmung: ruhig, hochwertig, cineastisch
Licht: weiches Fensterlicht, goldene Stunde, sanfte Schatten
Look: 35mm Film, feines Korn, geringe Schärfentiefe
Setting: reale, minimalistische Interiors (Naturmaterialien, Leinen, Stein)
Komposition: Subjekt klar zentriert, viel Negativraum
Farbwelt: warme, gedeckte Töne (sand, creme, amber)
Kein Text, keine Logos, keine künstlich wirkenden Reflexe.
Format: 4:5 (Feed) bzw. 9:16 (Story/Reel).
```

## 2. Produktclip / Szenen-Direction (Video) , Stil „Seedance"

```
Erzeuge ein 8-Sekunden-Video aus diesem Start-Frame.
Szene 1 (0,2s): Nahaufnahme [Produkt], weiches Licht zieht über die Oberfläche
Szene 2 (2,4s): langsame Kamerafahrt, Tropfen/Textur in Bewegung
Szene 3 (4,6s): Produkt in der Hand / In-use, dewy skin
Szene 4 (6,8s): ruhiger Hero-Frame mit Negativraum
Stil: cineastisch, slow motion, warm, premium. Keine Schnitte sichtbar.
```

## 3. UGC-Werbeskript (für Arcads / AI-Schauspieler) , Stil Kayvon/Arcads

```
Du bist Performance-Creative-Stratege für Beauty/Wellness.
Marke: [Name] | Produkt: [Produkt] | Zielgruppe: [z.B. Frauen 25,40, Skincare-affin]
Schreibe 5 UGC-Ad-Skripte (je 20,30 Sek) im Format:
- HOOK (1 Satz, stoppt den Scroll)
- BODY (Problem → Produkt → Beweis, gesprochen, locker, du-Ansprache)
- CTA (klar, ein Handlungsschritt)
Tonfall: nahbar, ehrlich, kein Werbe-Sprech.
Gib danach jedes Skript als Arcads-Prompt aus (actor type, scene, action, dialogue).
```

## 4. Carousel-Maker (als Claude-Projekt-Systemprompt) , Stil Kayvon

> In Claude → Projekte → neues Projekt „Carousel Maker" → diesen Text in die Projekt-Beschreibung. Danach nur noch Thema eintippen.

```
Du bist mein Karussell-Generator für Instagram (@sola.makes, AI-Creative-Studio Beauty/Wellness).
Wenn ich ein Thema gebe, baue ein 6,8-Slide-Karussell:
- Slide 1: starker Hook (Neugier oder Gegen-den-Strich-These), max 7 Wörter
- Slides 2,n: je 1 Idee, konkret, kurz, mit Beispiel
- vorletzte Slide: Mini-Case/Beweis ("kein Set, kein Shooting")
- letzte Slide: CTA "Kommentiere [WORT], ich schick dir [Asset]"
Stil: lowercase, ruhig, selbstbewusst. Keine Floskeln.
Gib zusätzlich Caption + 6 Hashtags (Nische + Tools) aus.
```

## 5. Caption + Lead-Magnet-Generator

```
Schreibe eine Instagram-Caption für @sola.makes zu diesem Post: [kurz beschreiben].
Struktur:
1) Hook-Zeile (Neugier/Unglaube), lowercase
2) 2,3 Zeilen Story/Beweis ("kein set, keine crew", ggf. Zahl)
3) Lead-Magnet-CTA: "kommentiere [EIN WORT], dann schick ich dir [Workflow/Prompt/Set]"
4) 6 Hashtags: Mix aus Beauty + AI + Tool-Namen
Tonfall: ruhig, hochwertig, nahbar. Max 600 Zeichen.
```

## 6. AI-Website bauen (Claude-Code-Stack) , Stil Valeri Sabev

```
Baue eine einseitige Premium-Landingpage für "sola.makes , KI-Creative-Studio für Beauty & Wellness".
Stack-Vorgaben:
- sauberes, modernes Layout (großzügiger Weißraum, edle Typo)
- Hero mit starker Headline + Sub + CTA
- Sektionen: Portfolio-Grid, "So entsteht's" (3 Schritte), Leistungen, Über, Kontakt/Buchung
- dezente Animationen (Fade/Slide beim Scrollen), schnelle Ladezeit
- Farbwelt: warm, gedeckt (sand/creme/amber), viel Schwarz/Weiß-Kontrast
Liefere alles als EINE index.html (HTML+CSS+JS inline), responsive, ohne externe Abhängigkeiten.
```

## 7. Positionierung & Angebot (Personal-Brand-Prompt) , adaptiert von Tyler Payne

```
Du bist Experte für Personal Branding, AI-Leverage und Angebots-Entwicklung.
Führe mich Schritt für Schritt zum profitablen Studio-Brand auf Instagram.
Meine Infos:
- Können/Expertise: AI-Produktfotografie & -video für Beauty/Wellness
- Stand auf Instagram: [Follower]
- Umsatzziel/Monat: [Ziel]
SCHRITT 1: 3 profitable Positionierungs-Winkel (Zielgruppe, Problem, Zahlungsbereitschaft).
SCHRITT 2: ein konkretes Angebot/Paket je Winkel (Leistung, Preisanker, Ergebnis).
SCHRITT 3: 30-Tage-Content-Plan, der genau dieses Angebot verkauft.
```
